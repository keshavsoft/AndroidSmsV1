AndroidSmsV1

Prerequisites

  • Android Studio (latest version)
  • Internet connection (for Gradle)
  • Android SDK
  • Real Android device (SMS does not work on emulator)

Setup & Run Project

Clone Repository

git clone https://github.com/keshavsoft/AndroidSmsV1.git

Open in Android Studio

  • Open Android Studio
  • Open → Existing Project
  • Select AndroidSmsV1 folder

Gradle Sync

  • Wait until Gradle Build Finished
  • First build may take 5–10 minutes

Connect Device

  • Enable Developer Options
  • Enable USB Debugging
  • Connect phone via USB

Permissions

Ensure the following permissions exist in AndroidManifest.xml:

  • READ_SMS
  • SEND_SMS
  • RECEIVE_SMS

Run App

  • Select device
  • Click ▶ Run
  • Grant runtime permissions

App Testing

Inbox

  • View received SMS
  • Sender number, message, date/time

Send SMS

  • Enter phone number
  • Type message
  • Press Send

Sent Messages

  • View sent SMS history

Troubleshooting

  • File → Sync Project with Gradle Files
  • Build → Clean Project
  • Build → Rebuild Project
